Quick Linux System Analysis

Does your system monitoring ever look like this?

What tools do you have to perform a quick analysis of what is going on?

uptime

How long has the server been running and current load

dmesg

Shows system messages

mpstat -P ALL 1

Shows CPU utilization

iostat -xz 1

Shows Disk I/O utilization

sar -n DEV 1

Show network interface utilization
